Instruction on how to set up and use.

The fisrt thing you will need to do is include the vector.js and shape.js into your project. All shapes are fully documented. Every shape has an update() and render() method that you can use the update() to set all the properties on each frame and the render() will take care of drawing the shape with those new values.

You can also check the app.js for a quick view on how those shape on top are coded.

Creating a ball

        
        var ball = Ball(position:Vector, radius:number, color:string, context:CanvasRenderingContext2D);

Creating a circle

        
            var circle = Circle(position:Vector, radius:number, hasFill:boolean, fillColor:string hasStroke:boolean, strokeColor:string, lineWidth:number, context:CanvasRenderingContext2D);
